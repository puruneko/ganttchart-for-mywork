# ズームスクロール感度の調整

## Status

closed

---

## Summary

ズームがmajor日・minor時間のとき、スクロール量に対してズームイン/アウトの変化が少なく、大量のホイール操作が必要でユーザー体験が悪い。各ズームレベルで適切なスクロール量でズームレベルが遷移するよう設定を調整する。

---

## Current Direction

`zoom-gesture.ts` のホイールイベントで `zoomFactor` が `0.95`/`1.05`（5%変化）に全ズームレベル固定されている。スケール適応型のズームステップに変更し、ズームレベルに応じた適切な感度を実現する。

設計：

```
scale >= 5.0 のとき: ステップ 12%（高倍率側 ~9回で次レベルへ）
scale >= 1.0 のとき: ステップ  8%（中倍率側 ~10回で次レベルへ）
scale >= 0.1 のとき: ステップ  6%（低倍率側 ~12回で次レベルへ）
それ以外:            ステップ  4%（超低倍率）
```
